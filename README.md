# Jornada-Dados
**Análise Preditiva de Sobrevivência no Titanic com Random Forest:**
- Problema: Prever a sobrevivência de passageiros com base em dados socioeconômicos.
- Ferramentas: Python, Pandas, Seaborn, Scikit-Learn.
- Destaque Técnico: Tratamento de dados nulos por mediana e criação de novas variáveis (Family Size).
- Resultado: Modelo com 81,56% de acurácia.
